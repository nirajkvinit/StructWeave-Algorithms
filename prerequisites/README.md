# Prerequisites

This directory contains foundational knowledge required before diving into algorithmic patterns and problem-solving. These materials ensure you have the essential building blocks for tackling coding interview problems effectively.

## Who Should Start Here

- Beginners starting their algorithm journey
- Self-taught programmers looking to fill knowledge gaps
- Anyone wanting to review fundamentals before advancing to patterns

## Content Overview

### Foundational Skills

Build your core problem-solving abilities:

- [Programming Basics](programming-basics.md) - Variables, control flow, functions, and fundamental constructs
- [Computational Thinking](computational-thinking.md) - Problem decomposition, pattern recognition, and abstraction
- [Debugging Strategies](debugging-strategies.md) - Systematic approaches to finding and fixing bugs

### Complexity Analysis

Understand how to analyze algorithm efficiency:

- [Time Complexity](time-complexity.md) - Big O notation and runtime analysis
- [Space Complexity](space-complexity.md) - Memory usage and auxiliary space analysis

### Data Structures

Master the fundamental data structures:

- [Arrays and Strings](arrays-and-strings.md) - Contiguous memory, indexing, and string manipulation
- [Linked Lists](linked-lists.md) - Node-based structures, pointers, and traversal
- [Stacks and Queues](stacks-and-queues.md) - LIFO and FIFO operations
- [Hash Tables](hash-tables.md) - Hashing, collision resolution, and O(1) lookups
- [Trees](trees.md) - Hierarchical structures, traversals, and binary trees
- [Graphs](graphs.md) - Vertices, edges, representations, and traversals
- [Heaps](heaps.md) - Priority queues and heap operations
- [Tries](tries.md) - Prefix trees and string-based retrieval

### Mathematical Foundations

- [Number Theory](number-theory.md) - GCD, primes, modular arithmetic, and divisibility

## Suggested Learning Path

For optimal progression, work through the materials in this order:

1. **Start with fundamentals**
   - Computational Thinking
   - Programming Basics
   - Debugging Strategies

2. **Learn complexity analysis**
   - Time Complexity
   - Space Complexity

3. **Linear data structures**
   - Arrays and Strings
   - Linked Lists
   - Stacks and Queues

4. **Associative structures**
   - Hash Tables

5. **Hierarchical structures**
   - Trees
   - Heaps
   - Tries

6. **Graph structures**
   - Graphs

7. **Mathematical concepts** (as needed for specific problems)
   - Number Theory
